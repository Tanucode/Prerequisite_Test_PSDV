Question 2: Mutable vs. Immutable Data Types:

  • Explain the difference between mutable and immutable data types in Python. Provide examples of each

Immutable Data Types

The Immutable data types in python are those that can never change their values in place.This means that the one particular memeory address will be storing only one value at a time. If a varibale is assigned to these data types , then these values can not be changed during the whole execution of program.

In python variable neames are just the references to the value-objects, that is, data values. The variable name do not store values themselves, they are not storage containers.variables act as references to objects, not as containers that directly hold values. This is why when we assign a value to a variable, we actually assigning a reference to the memory location where the value is stored. The immutable data types in Python are those that cannot be modified after they are created. If an operation tries to modify the value, it will create a new object, leaving the original object unchanged.

  • The Immutable data types in python includes:

    (1) Integers

    (2) Floating Point Numbers

    (3) Strings

    (4) Tuples

    (5) Bools

Mutable Data Types

The mutable data types are those whose values can be changed in place. This means that objects of mutable types can have their values modified after they are created. Unlike immutable types, when a mutable object is modified, the change affects the original object in memory.

Thses are mutable data types in python:

(1) Lists

import matplotlib.pyplot as plt

def generate_fibonacci(n):
    if n <= 0:
        return []
    fibonacci = [0, 1]
    for _ in range(2, n):
        fibonacci.append(fibonacci[-1] + fibonacci[-2])
    return fibonacci[:n]

def plot_fibonacci(n):
    fibonacci = generate_fibonacci(n)
    plt.plot(range(1, n + 1), fibonacci, marker='o')
    plt.title("Fibonacci Sequence")
    plt.xlabel("Index")
    plt.ylabel("Fibonacci Number")
    plt.grid()
    plt.show()

    cleaned = ''.join(char.lower() for char in s if char.isalnum())

from statistics import mean, median
import math

def remove_duplicates(input_list):
    unique_elements = []
    seen = set()
    for element in input_list:
        if element not in seen:
            unique_elements.append(element)
            seen.add(element)

    mean_value = mean(unique_elements) if unique_elements else None
    median_value = median(unique_elements) if unique_elements else None

    return unique_elements, mean_value, median_value

input_list = [4, 5, 6, 4, 2, 5, 3, 2, 8]

def anagrams(word, word_list):
    sorted_word = sorted(word)
    anagrams = [w for w in word_list if sorted(w) == sorted_word]

import matplotlib.pyplot as plt
from collections import Counter
import re

def read_file(file_path):
    with open(file_path, 'r') as file:
        text = file.read()
    return text

def count_words(text):
    words = re.findall(r'\b\w+\b', text.lower())
    word_count = len(words)
    return word_count, words

def most_frequent_word(words):
    word_counter = Counter(words)
    most_common_word, frequency = word_counter.most_common(1)[0]
    return most_common_word, frequency, word_counter

def visualize_word_frequencies(word_counter):
    words, frequencies = zip(*word_counter.items())
    plt.figure(figsize=(10, 6))
    plt.bar(words, frequencies)
    plt.xlabel('Words')
    plt.ylabel('Frequency')
    plt.title('Word frequency')
    plt.xticks(rotation=45)
    plt.show()

def main(file_path):
    text = read_file(file_path)
    word_count, words = count_words(text)
    print(f"Total number of words: {word_count}")
    print(f"The words are:-{words}")

    most_common_word, frequency, word_counter = most_frequent_word(words)
    print(f"The most frequent word is '{most_common_word}' with {frequency} occurrences.")

    visualize_word_frequencies(word_counter)

import matplotlib.pyplot as plt

def normalize_data(numbers):
    min_val = min(numbers)
    max_val = max(numbers)
    normalized_numbers = [(x - min_val) / (max_val - min_val) for x in numbers]
    return normalized_numbers
